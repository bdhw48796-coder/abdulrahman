HTML Basics

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It provides the structure and content of a webpage, allowing developers to define headings, paragraphs, links, images, and other elements. and it is used to create the structure and content of web pages. HTML consists of a series of elements represented by tags, which define the different parts of a webpage. These elements can include headings, paragraphs, lists, links, images, forms, and more.

HTML Elements

HTML elements are the building blocks of a webpage. They are represented by tags, which are enclosed in angle brackets. Each element typically consists of an opening tag, content, and a closing tag. For example, the

element is used to define a paragraph:

<p>This is a paragraph.</p>

HTML elements can also have attributes, which provide additional information about the element. For example, the element is used to create hyperlinks, and it can have an href attribute that specifies the URL of the linked page:

<a href="https://www.example.com">Visit Example</a>

HTML Structure

HTML documents follow a specific structure that includes a doctype declaration, an html element, a head element, and a body element:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

The doctype declaration specifies the HTML version being used. The html element is the root element that wraps the entire document. The head element contains meta information, such as the character encoding and title of the page. The body element contains the visible content of the webpage.

HTML Paragraphs

HTML paragraphs are defined using the

element. Paragraphs are used to group related sentences together and create blocks of text. For example:

<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

Paragraphs can also contain other inline elements, such as links, images, and formatting tags. For example:

<p>This is a paragraph with a <a href="https://www.example.com">link</a> and an <em>emphasized</em> word.</p>

HTML Lists

HTML provides two types of lists: ordered lists and unordered lists. Ordered lists are used to create numbered lists, while unordered lists are used to create bulleted lists. For example:

<h3>Ordered List</h3>
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Unordered List

<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>

Lists can also be nested, allowing for the creation of hierarchical structures. For example:

<ul>
    <li>Item 1</li>
    <li>Item 2
        <ul>
            <li>Subitem 2.1</li>
            <li>Subitem 2.2</li>
        </ul>
    </li>
    <li>Item 3</li>
</ul>

HTML Images

HTML images are used to display pictures and graphics on a webpage. Images are defined using the element, which requires the src attribute to specify the image source and the alt attribute to provide alternative text for accessibility. For example:

<img src="image.jpg" alt="Description of the image">

Images can also have additional attributes such as width and height to control their size, as well as CSS styles for further customization.

HTML Semantic Elements

HTML5 introduced semantic elements that provide meaning to the structure of a webpage. These elements help improve accessibility and search engine optimization (SEO) by clearly defining the purpose of different sections of content. Some common semantic elements include:

  • : Represents the introductory content or a group of navigational links.
  • : Represents a section of navigation links.
  • : Represents the main content of the document.
  • : Represents a thematic grouping of content.
  • : Represents a self-contained piece of content, such as a blog post or news article.
  • : Represents content that is tangentially related to the main content, such as sidebars or pull quotes.
  • : Represents the footer of a document or section, typically containing copyright information or links.

Using semantic elements helps create a more meaningful and accessible web experience for users and search engines alike.
